# Contributing

Thanks for helping improve Herdr Server Aware.

## Development

```bash
cargo fmt
cargo clippy -- -D warnings
cargo test
cargo build --release
```

Keep changes small and document user-facing behavior in `README.md`.

## Pull requests

- Explain the user-visible change.
- Add or update tests for non-trivial logic.
- Update `CHANGELOG.md` under `Unreleased`.
